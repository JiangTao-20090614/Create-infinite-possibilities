# Earthquake Real-Time Visualizer

USGS API 实时获取全球地震数据，并用 Python + Plotly 绘制交互式地球分布图。

## 功能亮点
- **自动获取数据**：无需手动下载 CSV，直接从 USGS API 拉取最新地震记录
- **交互地球图**：可旋转、缩放、悬停查看震级、地点、深度、时间
- **筛选支持**：默认显示 mag ≥ 5.0 的强震
- **健壮性**：动态路径、错误捕获、调试输出
- **高清保存**：交互图保存为 HTML 文件，可离线分享
